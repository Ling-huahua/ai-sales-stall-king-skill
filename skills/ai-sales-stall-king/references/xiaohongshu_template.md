# 小红书种草/Plog/视频模板

## 输出格式

普通图文笔记：

```
📌 **标题**：[数字]+[情绪词]+[悬念/反差]

---

[正文，用真实体验感的口吻]

---

🏷️ [标签1] [标签2] [标签3] [标签4]

📷 配图建议：[具体拍摄指导]
```

Plog 笔记：

```
📌 **Plog标题**：[生活场景]+[情绪/痛点]+[轻种草钩子]

### Plog图片组
1. 第1张：生活感封面/今日分享感
2. 第2张：商品进入生活场景
3. 第3张：材质/细节真实近景
4. 第4张：使用/搭配/摆放过程
5. 第5张：卖点说明小卡
6. 第6张：互动收尾/评论引导

### Plog正文
[像日常记录，不像广告；每段短一点，有生活碎片感]

🏷️ [标签1] [标签2] [标签3] [标签4]
```

小红书带货视频：

```
📌 **视频标题**：[痛点/反差/结果钩子]

### 视频封面
[封面字 + 画面建议]

### 完整口播文案
[45-60秒可直接照读版]

### 视频分镜
| 时间 | 画面 | 镜头/动作 | 口播 | 字幕重点 |

### 拍摄参数
[比例、时长、机位、灯光、背景、收音、手机倍率、B-roll清单]

### 发布建议
[标题、标签、评论区引导]
```

## 标题公式

- "用了[X]天，我把[Y]块的[竞品]扔了"
- "在[某地]发现了这个[产品]，直接[效果]"
- "[价格]块的[产品]，效果完全不输[大牌/贵价品]"
- "[谁]都在偷偷用的[产品]，我终于找到了"

## 正文结构

### 开头（1-2句）：用痛点/场景勾住人
- "姐妹们，[痛点场景]真的谁懂啊"
- "之前一直[困扰]，直到我发现[产品]"
- 开头必须让目标用户立刻代入“这不就是我吗”，不要先介绍产品参数

### 中间（3-5句）：真实体验+产品亮点
- 用"我用了之后"的视角
- 以“亲自试过后真心推荐”的姿态写，不要旁观解说图片
- 不要说"这个产品有X功能"，要说"我用的时候发现..."
- 每个亮点配一句感受
- 把材质、结构、功能转换成用户利益：舒服、省心、不尴尬、显质感、少踩雷、想复购

### 结尾（1-2句）：互动钩子
- "你们[相关场景]的时候会[怎么做]吗？"
- "评论区告诉我[问题]，我帮你们[解决]"

## 语气要求

- 像姐妹/朋友推荐，不是广告
- 语气要确定、有偏爱、有推荐欲，像“这个我真想安利给你”
- 口语化，用"真的""姐妹们""绝了"等
- 偶尔用 emoji 增加亲和力但不过度
- 普通小红书正文遵循主规则，不少于350字；只有用户明确要求短版、Plog碎片段落或评论区补充时，才可控制在150-300字

## 禁用语态

不要在面向消费者的小红书文案里写：

- “看原图细节更像”
- “看实物细节是”
- “看起来像”
- “应该是”
- “疑似”
- “视觉推断”
- “从图片看”
- “我不确定”

这些话会让用户觉得你在解说图片、甩锅、不敢推荐。正确写法是把观察转成该品类的体验和利益：

- 不写：“这款看实物细节是[某材质/某结构]”
- 写：“用起来最明显的是[体验利益]，解决了我之前[具体痛点]”
- 不写：“看起来像[某功能]”
- 写：“实际使用时，[功能/结构]带来的好处是[可感知结果]”

未确认的成分、认证、尺码等硬参数不要编；可以不写，或放到末尾提示“尺码/成分下单前可确认”。

## 标签策略

- 1-2个品类大词（如 #好物分享 #平价好物）
- 1-2个场景词（如 #夏天必备 #通勤神器）
- 1个情绪词（如 #相见恨晚 #后悔没早买）

## 配图建议格式

- 主图怎么拍（什么角度、什么光线、什么背景）
- 细节图拍什么（展示哪个功能点）
- 对比图怎么拍（和什么对比、突出什么）
- 建议使用自然光、生活场景

## Plog 生成要求

Plog 不是普通商品海报合集，而是“生活记录里顺手种草”。生成 Plog 时必须做到：

- **生活感优先**：像用户真实一天里的记录，不要每张都像广告图。
- **图片顺序有故事**：先场景/情绪，再商品出现，再细节，再使用/搭配，再卖点说明，最后互动。
- **低广告感**：文字标注少一点，像手账/便签/随拍说明，不要满屏大字促销。
- **原图真实性**：用户给的实拍图能用就优先裁切/增强；AI补图只补生活场景、构图和说明，不要改商品款式、颜色、材质和细节。
- **适合多图输入**：多张原图时，主图做封面或商品出现页，细节图做真实信任页，包装/标签图做说明页，使用/场景图做代入页。
- **敏感品类合规**：贴身衣物、美妆、食品、儿童用品等品类，不要低俗展示、不要夸大功效。

### Plog 图片组结构

默认输出 6 张，可按用户需求调整为 4-9 张：

1. **封面生活感图**：场景化标题，不像硬广；标题 8-14 字。
2. **商品自然出现图**：商品在桌面/包里/衣柜/梳妆台/通勤场景中出现。
3. **真实细节图**：材质、做工、包装、结构细节，优先用原图裁切增强。
4. **使用/搭配图**：展示怎么用、怎么搭、适合什么场景。
5. **卖点说明图**：最多 3 个卖点，短句，不堆参数。
6. **互动收尾图**：评论关键词、尺码/价格/细节咨询引导。

### Plog 文案结构

- 开头：像日常记录，例如“今天整理东西发现这个真的用得太频繁了……”
- 中间：用生活碎片带出 2-4 个卖点，不要参数堆砌。
- 结尾：轻互动，例如“想看细节/尺码/价格我放评论区”。
- 语气：比普通种草文更松弛、更像日记，但仍然要有推荐欲。

## 小红书带货视频要求

小红书视频和抖音不同，不要过度喊麦或强逼单。它更适合“真实分享 + 细节证明 + 温和转化”。

必须输出：

1. **封面方案**：封面字、画面主体、商品位置、标题排版。
2. **前3秒钩子**：痛点、反差、结果承诺或避坑提醒。
3. **完整口播文案**：45-60 秒，可直接照读。
4. **视频分镜表**：时间、画面、镜头/动作、口播、字幕重点。
5. **拍摄参数**：9:16、时长、机位、景别、灯光、背景、收音、手机倍率/焦段。
6. **B-roll清单**：产品全景、细节近景、使用/搭配、包装、价格/咨询引导。
7. **主播动作卡**：每镜站位、手部动作、眼神方向、切换点。
8. **发布建议**：标题、标签、评论区引导、置顶评论。

### 小红书视频镜头默认结构

| 时间 | 镜头目标 | 画面建议 |
|---|---|---|
| 0-3秒 | 抓痛点 | 真人/手部 + 痛点大字 |
| 3-8秒 | 商品出现 | 商品全景或原图画中画 |
| 8-18秒 | 核心卖点1 | 材质/功能/效果细节 |
| 18-30秒 | 核心卖点2 | 使用场景或搭配场景 |
| 30-42秒 | 信任细节 | 包装、做工、真实细节、对比 |
| 42-55秒 | 温和转化 | 适合人群 + 评论/私信引导 |

### 小红书视频语气

- 像认真分享，不像直播叫卖。
- 多用“我会推荐给…”“这个细节我挺喜欢…”“适合…”。
- 少用“赶紧买”“不买亏”“最后一天”等强促销话。
- 未确认的价格、功效、材质成分不要编。
