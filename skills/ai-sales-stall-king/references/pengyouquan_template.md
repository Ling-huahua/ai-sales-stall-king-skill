# 朋友圈文案模板

## 核心理念

朋友圈卖货的最高境界：**不像在卖货，像在分享好东西。**

用户刷朋友圈时的心理防线比刷小红书高——他们知道这是广告。所以文案必须更软、更真。

写朋友圈时要站在“我真的喜欢这个东西，所以顺手推荐给朋友”的位置。不要写成“我看图片/看实物像什么”的验货语气，也不要用“应该、疑似、看起来”削弱信任。未确认的硬参数不写进正文，正文只写确定的生活场景、感受和推荐理由。

## 输出格式

提供两个版本：

### 短版（1-2句）
适合配一张好图，随手一发，不费力。
```
[一句话感受/发现]
[轻描淡写的一句 + 留白让人主动问]
```

### 长版（带故事）
适合展示产品的来龙去脉，建立信任。
```
[个人故事/场景引入]

[产品出现 + 为什么推荐]

[使用感受/效果]

[轻描淡写的收尾，不硬推]
```

## 短版公式

- "[某个瞬间]的时候，发现[产品]真的好用"
- "[谁]给我推荐了[产品]，用了之后[感受]"
- "最近[做了什么]，全靠[产品]撑着"
- "[一句话感受]。懂的都懂。"

## 长版结构

### 开头：个人场景/故事
- 不说产品，先说自己的生活/工作/经历
- 制造共鸣（"最近真的太忙了""天热到不行"）

### 转折：产品自然出现
- "刚好""正好""突然想到"等自然引入
- 不刻意推销，像顺手一提
- 语气确定，有偏爱感，像自己真的愿意留着用/推荐给朋友

### 感受：真实体验
- "用了一段时间""说实话""挺意外的"等真实表达
- 优点实话实说，小缺点也提一点（增加可信度）

### 收尾：轻描淡写
- 不催促购买，不做紧迫感
- "有需要的可以找我""感兴趣的我细说"
- 留白让人主动来问

## 配图建议

- 使用场景图（自己正在用的照片）> 产品摆拍
- 生活感强，不要太"卖家秀"
- 1张好图比9张普通图效果好
